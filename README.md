## Build Setup

```bash
#firstly, check your node and npm version. in terminal add:

node -v 
#should be higher, than 14
npm -v 
#should be equal 8, or higher

#after that, run the following below :

# install dependencies

$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
